# Examples

Scripts under `examples/` demonstrate UQ-ADF reconstructions and plotting utilities.
Plots are written to `examples/plots/` (gitignored).

Run examples:
- `./venv/bin/python examples/uq_adf_smoke.py` (quick sanity check)
- `./venv/bin/python examples/uq_adf_plots.py` (convergence sweeps, heatmaps)
- `./venv/bin/python examples/uq_adf_darcy_2d.py` (2D Darcy reconstruction)
  - add `--fast` for reduced resolution and quicker diagnostics
